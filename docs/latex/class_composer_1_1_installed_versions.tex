Lets you reload the static array from another file

This is only useful for complex integrations in which a project needs to use this class but then also needs to execute another project\textquotesingle{}s autoloader in process, and wants to ensure both projects have access to their version of installed.\+php.

A typical case would be PHPUnit, where it would need to make sure it reads all the data it needs from this class, then call \mbox{\hyperlink{class_composer_1_1_installed_versions_ad40c96bf7223486a1e6d077ea4582f87}{reload()}} with {\ttfamily require \$\+CWD/vendor/composer/installed.php} (or similar) as input to make sure the project in which it runs can then also use this class safely, without interference between PHPUnit\textquotesingle{}s dependencies and the project\textquotesingle{}s dependencies.
